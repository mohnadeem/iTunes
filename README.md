# iTunes Search
A native iOS app displaying the search results.

# Highlights
* VIPER Design pattern
* Cocoapods
* Alamofire
* Kingfisher

# Requirements
* iOS 11.0+
* A Mac running macOS Big Sur
* Xcode 12+
* Swift 5+

# How to Run
Use XCode to run, build and test the code:
* Select Product > Run from the menu to run the app
